Etiquetas HTML - mi referencia
Etiqueta Para que sirve Ejemplo
p Para poner párrafos 

                <p>Párrafo diferente</p>
br Para saltos de línea 

                Hola<br>
                Cómo estás?
hr Para poner una línea recta horizontal, separa temática 

                Saludo
                <hr> 
                Despedida
code Para poner código 

                <code>

                #include <stdio.h>;
                int main() {
                    printf("Hola mundo");
                    return 0;
                }
                </code>
pre Para poner, citar el texto como está,con el formato ya definido 

                <pre>
                Párrafo 
                como quieras
                </pre>
h# Para poner encabezado jerarquicamente 

                <h1>Encabezado diferente</h1>
title Para poner título descriptivo 

                <title>Título diferente</title>
head Para poner metadatos del navegador 

                <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                </head>
body Para poner el contenido principal de la página 

                <body>
                <p>Contenido de la página</p>
                </body>
ol Para poner listas ordenadas 
                1. Uno
                2. Dos
                3. Tres
ul Para poner listas desordenadas 
                - Saludo
                - Despedida
                - Pregunta
dl Para poner listas de definición 
                HTML
                  Lenguaje de marcado
                CSS
                  Lenguaje de estilos
img Para poner imágenes 

                <img src="imagen.jpg" alt="Descripción de la imagen">
label Para poner etiquetas para elementos de formulario 
                <label for="campo1">Nombre:</label>

                Nombre:
input Para poner campos de entrada en formularios 

                <input type="text" id="campo1" name="Nombre">
